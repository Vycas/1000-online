About the project

This project was started as part of Python course at my university. I needed the topic for the project so I have chosen this game because I know and like it a lot. My friends and I play this game very often during breaks and lectures. We find it very addictive. So I realized that it would be great idea to be able to play it online. This way we could play anywhere! And finally I could deliver this game to masses of people.

About technologies

This project is written in Python for all server-side computation. Client-side is written in JavaScript for game dynamics and communications with server. JSON is used for communication format because of its simplicity and for being lightweight. I decided to use Google charts to show the progress of the game. Finally everything is hosted on Google Apps Engine because its free and offers a lot of resources.
